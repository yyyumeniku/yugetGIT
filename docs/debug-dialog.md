# Debug Dialog

Use this command to manually open the pre-launch setup dialog while in-game:

`/backup debug-dialog`

## What it is for
- Testing the Git/Git-LFS bootstrap UI.
- Re-checking missing dependency flows.
- Verifying first-run dialog behavior without restarting the whole game client.

## Notes
- This command is diagnostic and safe to run.
- It does not create a backup by itself.
